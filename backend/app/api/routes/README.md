# API Routes

REST endpoint tanımları bu dizinde yer alacaktır.

## Planlanan Route Modülleri

- `auth.py` — Kayıt, giriş, token yenileme
- `jobs.py` — İlan CRUD
- `cvs.py` — CV yükleme ve yönetimi
- `applications.py` — İlan-CV eşleştirme
- `agents.py` — AI ajan endpoint'leri
- `dashboard.py` — Özet istatistikler
