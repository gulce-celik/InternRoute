# React Frontend — InternRoute

Vite + React + TypeScript tabanlı frontend uygulaması.

## Kurulum

Sprint 1'de tamamlanacak. Bkz. kök `README.md`.

## Planlanan Sayfalar

| Sayfa | Route | Açıklama |
|-------|-------|----------|
| Login | `/login` | Giriş |
| Register | `/register` | Kayıt |
| Dashboard | `/` | Başvuru özeti ve istatistikler |
| Jobs | `/jobs` | İlan listesi ve ekleme |
| Job Detail | `/jobs/:id` | İlan detayı + analiz |
| CVs | `/cvs` | CV yönetimi |
| Applications | `/applications` | Başvuru eşleştirmeleri |
| Cover Letter | `/applications/:id/cover-letter` | Niyet mektubu üretimi |
| Mock Interview | `/interview/:id` | Mock mülakat oturumu |
