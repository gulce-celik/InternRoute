# Services

İş mantığı katmanı — API route'larından ayrılmış servis sınıfları.

## Planlanan Servisler

- `auth_service.py` — Kullanıcı kayıt/giriş
- `job_service.py` — İlan CRUD işlemleri
- `cv_service.py` — CV yükleme, PDF parsing
- `application_service.py` — Başvuru eşleştirme
- `file_service.py` — Dosya depolama yönetimi
- `dashboard_service.py` — İstatistik hesaplama
