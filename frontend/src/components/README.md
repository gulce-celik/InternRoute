# Components

Yeniden kullanılabilir UI bileşenleri.

## Planlanan Bileşenler

- `Layout/` — Ana sayfa düzeni, navbar, sidebar
- `JobCard/` — İlan kartı
- `CVUploader/` — PDF yükleme bileşeni
- `ApplicationStatus/` — Başvuru durumu badge
- `AnalysisReport/` — Analiz sonuç görünümü
- `CoverLetterEditor/` — Cover letter düzenleme
- `InterviewChat/` — Mock mülakat sohbet arayüzü
- `ui/` — Temel UI primitives (Button, Input, Modal)
