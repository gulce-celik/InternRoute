# Models

SQLAlchemy ORM modelleri.

## Planlanan Modeller

- `User` — Kullanıcı hesabı
- `Job` — İş/staj ilanı
- `CV` — Yüklenen CV versiyonu (PDF metadata)
- `Application` — İlan-CV eşleştirmesi ve başvuru durumu
- `InterviewSession` — Mock mülakat oturumu
- `InterviewAnswer` — Mülakat soru-cevap kayıtları
