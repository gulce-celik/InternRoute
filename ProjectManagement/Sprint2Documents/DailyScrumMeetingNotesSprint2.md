## Örnek standup notları (özet)

> Tam günlük diyaloglar Sprint 1 dosyasındaki gibi WhatsApp’tan export edilince buraya genişletilecek. Aşağıda sprint boyunca konuşulan ana hatlar var.

### 6 Temmuz 2026 — Sprint 2 planning
- **Gülce / Muhammed:** Sprint 2 backlog seçildi — US-2.1 (5), US-2.2 (5), US-2.3 (8), US-2.4 (8) = **26 puan**. US-2.1 job board ile başlıyoruz.

### 8–10 Temmuz — Job board (US-2.1)
- Job CRUD API, Board UI, status (applied / interview / offer / rejected), Pipeline strip, Dashboard job count.

### 11–13 Temmuz — CV locker (US-2.2)
- PDF upload API + storage, CVs sayfası, View PDF, silince pipeline’da CV boş kalır (yeniden seçilebilir).

### 14–16 Temmuz — Applications (US-2.3)
- Job + CV eşleştirme, notes + yazılı Q&A, status güncelleme, Application file paneli.

### 17–18 Temmuz — RAG foundation (US-2.4)
- PDF → text → chunk → embed → ChromaDB; `/memory/context`; CVs sayfasında technical preview.

### 19 Temmuz — Review + Retro
- Demo: Board → CV upload → Pipeline link → Dashboard stats. Sprint 3 AI ajanları Backlog’da.
